What is UI?

User Interface (UI) is everything users see and interact with on a screen. It includes buttons, forms, navigation, cards, modals, and all visual elements that make up a digital experience.

Great UI is clear, consistent, and accessible. It guides users effortlessly toward their goals while maintaining visual hierarchy and brand identity.

Through 10 hands-on projects, you'll build real mini-applications and learn UI principles by doing. Each project includes interactive teaching tips that explain design decisions in context.
